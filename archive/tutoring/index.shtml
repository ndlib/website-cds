Math Software Tutoring


The O'Meara Mathematics Library is pleased to announce that mathematics software tutoring will be available this fall semester. This service is free of charge.

Location: Center for Digital Scholarship Classroom (129 Hesburgh Library - 1st Floor Northeast)

Time: 30-minute slots, between 6 PM - 10 PM

Topics

  • Sundays: Matlab
  • Mondays: R
  • Tuesdays: LaTeX/BibTeX


To see scheduled tutoring dates and register, see the Math Software Tutoring Calendar. (For Math Tutoring information (algebra, calculus, finite math, etc.), please see the O'Meara Mathematics Library web site.)
