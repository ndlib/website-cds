Room 222

Make a Reservation <insert link>

Capacity: 25

Description: Room 220 is a collaborative instruction room for classes of up to 25 students. When not used for classes, it can be reserved for group meetings. The room is configured in five "pods." Each pod contains a table seating 4 or 5 chairs, a 42" monitor on a rolling stand, one laptop, and a switch box allowing 3 additional VGA connections for monitor sharing. Each pod is independent of the others (cross-pod screen-sharing is not available a this time). This room provides adequate sound control for normal conversation, but does not provide a secure environment for confidential discussions.

Note: Please plan on a 20-minute set-up session with technical support prior to your meting if you are a new user of this room.
